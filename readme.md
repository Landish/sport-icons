# Installation

```
bower install sport-icons --save
```

# Usage

```html
<link href="/bower_components/sport-icons/dist/css/sport-icons.css" />
```

```html
<i class="sport-icon tennis"></i>
<i class="sport-icon {sportname}"></i>
```
